AI Recommendation Logic

About Project

This project is a simple recommendation system made using Python.

How It Works

- User enters a movie genre.
- The program checks the genre.
- The program recommends movies based on the user's choice.

Features

- Takes user input
- Uses if-else conditions
- Gives movie recommendations
